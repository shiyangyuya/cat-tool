## 安装 {#installation}

::: code-group

```sh [npm]
$ npm install cat-tool
```

```sh [pnpm]
$ pnpm add cat-tool
```

```sh [yarn]
$ yarn add cat-tool
```

:::

## 使用示例 {#usage}

```js
import { Watermark } from "cat-tool";

new Watermark({ content: "hello world!" });
```
